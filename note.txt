Flow 
- git clone {project}
- add file .env
- buka file .env ubah DBName dengan nama database local
- download laragon
- download pematon adminer custom github
- extract dan taruh pada folder laragon/www
- jalankan laragon buka browser adminer => adminer.test
- buka folder project masuk ke terminal
- jalankan composer update
- composer install
- php artisan key:generate
- php artisan migrate --seed
- insert data pada table services, service_galleries, dan service_categories
